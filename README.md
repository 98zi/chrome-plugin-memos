# chrome-plugin-memos

Boilerplate for Chrome extension using Vue.js and Webpack with hot-reload enabled.

Memos的chrome插件。

## 功能

在设置中中输入网站域名、以及 openId进行配置。

**已开发**

编辑，发布，归档，编辑历史笔记功能。

**未完成**

标签，搜索功能。

## use

1. 下载已打包好的 dist-zip 文件，解压拖拽到 chrome 扩展中。
2. `npm run build` 重新构建
